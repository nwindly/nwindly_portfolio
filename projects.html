Restaurant 362

Restaurant 362 is a web-based application modeled after an e-commerce platform, aimed at optimizing pick-up and online ordering processes. Technologies used were HTML and CSS for the front-end, while both Javascript and Firebase was implemented for both front-end and back-end functionalities. Time constraints for implementing expected deliverables and varying levels of technical skills among the team were challenges the team encountered throughout the development process. Regardless, the project ultimately provided a functional, accessible, and user-friendly interface for order and account management.
